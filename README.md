# Heliopause's 32blit Project

## Clone the repo

```sh
git clone https://github.com/AnthonyDiGirolamo/heliopause.32blit
cd heliopause.32blit

git submodule update --init
cd third_party/pico-sdk
git submodule update --init
cd ../../third_party/pico-extras
git submodule update --init
```
